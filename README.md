# 🌐 Internet Diagnostikasi Tizimi

## 📖 Loyiha haqida

Internet Diagnostikasi Tizimi foydalanuvchilarga internet ulanishi holatini tekshirish, tarmoq tezligini o‘lchash va muammolarni aniqlash imkonini beruvchi zamonaviy veb-platformadir. Tizim internet sifati haqida real vaqt rejimida ma’lumot beradi va foydalanuvchilarga tarmoqdagi nosozliklarni aniqlashda yordam beradi.

## 🚀 Asosiy imkoniyatlar

* ⚡ Internet tezligini o‘lchash
* 📥 Yuklab olish (Download) tezligini tekshirish
* 📤 Yuklash (Upload) tezligini tekshirish
* 📶 Ping va kechikishni aniqlash
* 🌍 IP manzilni ko‘rsatish
* 📍 Foydalanuvchi joylashuvini aniqlash
* 📊 Tarmoq holati bo‘yicha statistik ma’lumotlar
* 🔍 Internet muammolarini diagnostika qilish
* 📱 Barcha qurilmalar uchun moslashuvchan dizayn

## 🎯 Tizimning maqsadi

Ushbu platforma foydalanuvchilarga internet ulanishi sifati haqida aniq ma’lumot berish, tarmoqdagi muammolarni tez aniqlash va internet xizmatlari samaradorligini baholash imkonini yaratadi.

## 📊 Diagnostika natijalari

Tizim quyidagi ma’lumotlarni taqdim etadi:

* Internet tezligi
* Ping va Latency
* Jitter ko‘rsatkichlari
* Paket yo‘qotilishi (Packet Loss)
* IP manzil
* Internet provayderi ma’lumotlari
* Tarmoq barqarorligi

## 📈 Kelajakdagi rejalar

* Real vaqt monitoringi
* Internet sifati bo‘yicha hisobotlar
* Mobil ilova ishlab chiqish

## 📄 Litsenziya

Ushbu loyiha internet ulanishi sifatini monitoring qilish va diagnostika qilish maqsadida ishlab chiqilgan.
